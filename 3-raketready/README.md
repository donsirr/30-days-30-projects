# Raket-Ready (Philippine Freelancer Tax Utility)

A Next.js + Tailwind CSS application designed to help Filipino freelancers navigate tax compliance.

## Features

- **Tax Calculator**: Compare 8% Flat Rate vs. Graduated Rates.
- **Registration Roadmap**: Step-by-step guide for DTI, LGU, and BIR compliance.
- **Input Console**: Configure profession type, branding, and income source.
- **Dynamic Checklist**: Detailed list of requirements based on your profile.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS (v4)
- Framer Motion
- Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
